About the Project

The Heart Disease Detection Project is a machine learning project that uses a dataset of medical features to train a model that can predict the likelihood of heart disease in a patient. The project uses a combination of data preprocessing, feature engineering, and model selection to develop a highly accurate model.

The project is designed to help medical professionals diagnose heart disease more accurately and effectively, and to provide patients with a better understanding of their risk factors.

Project Details

The project uses a dataset of 820 samples, each with 14 medical features. The dataset is split into training and testing sets, with 70% of the data used for training and 30% used for testing.

The project uses a range of machine learning algorithms, including logistic regression, decision trees, and random forests, to develop a highly accurate model.
